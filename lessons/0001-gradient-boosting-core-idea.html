Gradient Boosting：不断修正当前模型

这一课只解决一个问题：Gradient Boosting 中，每一棵新树究竟在做什么？

先给结论

Gradient Boosting 先建立一个粗糙预测，然后不断增加小树，让每棵新树修正当前模型仍然存在的错误。

一个最小例子

假设要预测四辆车的价格，真实价格如下。初始模型暂时不看任何特征，对所有车都预测平均价格 125。

车辆真实价格 y当前预测 F₀(x)残差 y − F₀(x)
A100125−25
B120125−5
C130125+5
D150125+25

残差表示当前预测应该向哪个方向修正：

正残差：预测低了，需要往上修。
负残差：预测高了，需要往下修。

一轮 Boosting

1. 当前模型预测初始模型 F₀(x) 对所有车辆预测 125。
2. 计算当前错误计算每个样本的残差：真实值减去当前预测。
3. 新树学习修正训练 tree₁(x)，让它尽量预测这些残差。
新模型：F₁(x) = F₀(x) + η × tree₁(x)

η 是 learning rate。它控制本轮修正幅度，避免一棵树修得过猛。

为什么这能降低平方误差？

对第 i 个样本，真实值 yᵢ 已固定，当前所有树组合后的预测是 F(xᵢ)。使用带 ½ 的平方误差：

Lᵢ(F) = ½ × (yᵢ − F(xᵢ))²

这里对 当前预测值 F(xᵢ) 求导，因为新树能够改变的是预测值，而不是真实值。

∂Lᵢ / ∂F(xᵢ)
= ½ × 2(yᵢ − F(xᵢ)) × (−1)
= F(xᵢ) − yᵢ

残差定义为：

rᵢ = yᵢ − F(xᵢ)

因此：

− ∂Lᵢ / ∂F(xᵢ) = yᵢ − F(xᵢ) = rᵢ

所以在平方误差下，残差恰好就是 loss 对当前预测值的负梯度。如果平方误差不写 ½，负梯度会是 2rᵢ；方向相同，常数可以由 learning rate 吸收。

为什么沿负梯度方向会降低 loss？

导数表示：预测值 F 增加一点时，loss 会向哪个方向变化。

小幅修改 ΔF 后：
L(F + ΔF) ≈ L(F) + L′(F) × ΔF

如果选择：

ΔF = −η × L′(F)，其中 η > 0

那么 loss 的变化近似为：

L′(F) × ΔF = −η × [L′(F)]² ≤ 0

因此，小步沿负梯度方向修改预测，loss 不会上升，通常会下降。

Gradient Boosting 无法直接为每个样本单独保存一个修正值，所以训练一棵树去近似这些负梯度：

treeₘ(xᵢ) ≈ − ∂Lᵢ / ∂F(xᵢ)
平方误差下：treeₘ(xᵢ) ≈ rᵢ

然后把这棵树加入当前整体模型，再重新计算新的残差。每一轮修正的是当前所有树组合后的模型。

与 Decision Tree、Random Forest 的关系

Random Forest

很多树独立训练，最后平均。重点是让多棵树共同降低不稳定性。

Gradient Boosting

树按顺序训练，后一棵依赖前面模型的错误。重点是逐轮改进当前模型。

两者都使用 Decision Tree 作为基本组件，但组合树的方式不同。

理解检查

某样本真实值为 80，当前模型预测为 100。下一棵树应为它学习什么方向的修正？

本课压缩

当前整体模型 → 计算 loss 的负梯度 → 新树近似负梯度 → 加入新树 → 重复。平方误差下，负梯度恰好就是残差。
